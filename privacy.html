1. Visão geral

O Pay respeita sua privacidade. Esta política descreve quais dados o app coleta, por que são coletados e como são utilizados.

2. Dados coletados

  • Dados locais: transações, categorias e informações financeiras são armazenadas localmente no dispositivo para oferecer insights e gráficos.
  • Dados técnicos: informações básicas do dispositivo e eventos de uso para diagnóstico e melhoria do app (quando consentido/anonimizadas).

3. Uso dos dados

Os dados servem para gerar relatórios, detectar padrões de gasto e exibir sugestões. Não vendemos dados a terceiros.

4. Compartilhamento

Compartilhamos apenas com provedores necessários para funcionamento (ex.: serviços de análise anônima) e sempre com anonimização e conformidade legal.

5. Segurança

Aplicamos práticas padrão de segurança para proteger dados em trânsito e em repouso. Entretanto, nenhum método é 100% seguro.

6. Direitos do usuário

Você pode solicitar exclusão de dados ou informações sobre os dados coletados via suporte: grupodantas@gmail.com.

7. Contato

Para dúvidas sobre essa política, entre em contato: grupodantas@gmail.com
